
# Assistência em Foco — IFPB Campus Patos

Protótipo React + Vite para o portal "Assistência em Foco" com simulador de IVS e checklist inteligente.

## Como usar (local)

1. Instale dependências:
```bash
npm install
```

2. Rodar em desenvolvimento:
```bash
npm run dev
```

3. Build para produção:
```bash
npm run build
npm run preview
```

## Deploy no Vercel (recomendado)
1. Crie um repositório no GitHub e envie os arquivos.
2. Acesse https://vercel.com, conecte o repositório e clique em "Deploy".
3. Vercel detecta projeto Vite e faz o build automaticamente.

> Observação: este protótipo usa Tailwind; em ambiente de produção, verifique se as dependências estão corretas e execute `npm install` antes do deploy.
